Voting Application

A web-based Voting Application where users can register as voters, create elections, and cast votes securely. Admins can manage elections, voters, and candidates from an intuitive dashboard.

Features

1.User Features

a.Register as a Voter – Users can append themselves to the voter list by registering on the platform.
b.Login as Voter – Access your account using username and password.
c.View Eligible Elections – Only elections you are added to as a voter are visible.
d.Cast Votes – Select an election and vote for the candidates.
e.View Personal Info – Check your name, username, and logout when done.


2.Admin Features

a.Create Elections – Admins can create new elections by providing election details.
b.Admin Login – Access your admin panel using Election ID and password.
c.Add Head Voters – Add voters as head voters by providing their username.
d.Add Candidates – Register candidates and direct candidates for elections.
e.Manage Election Status – Activate or deactivate elections using a dedicated button.
f.Manage Elections – Delete elections via the “Manage” button.
g.View Results – Check the number of votes each candidate received.
h.View Voter List – See all voters added to the election.


Navigation Flow
Landing Page (index.html)
Options to Register, Login as Voter, or Create Election.

Voter Dashboard
Displays name, username, logout option, and list of eligible elections.

Clicking an election shows candidates and allows voting.

Admin Dashboard
Add head voters, candidates, and direct candidates.
Activate/inactivate elections.
Manage elections (delete elections).
View election results and voter list.


Technologies Used
Frontend: HTML, CSS, JavaScript,tailwind css
Backend: Node.js, Express.js
Database: MongoDB (or any other backend DB)







How to Use
Open index2.html in your browser.
Register yourself as a voter or login as a voter/admin.
Admin can create elections and manage voters/candidates.
Voters can select eligible elections and cast votes.
Check election results from the admin panel once voting is complete.
